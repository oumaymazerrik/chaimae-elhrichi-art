# Site vitrine Chaimae Elhrichi

Ouvrir `index.html` dans un navigateur.

## Pages

- `index.html` : accueil artistique
- `galerie.html` : galerie filtrable avec lightbox
- `apropos.html` : demarche et presentation
- `commander.html` : formulaire de commande
- `contact.html` : Instagram, WhatsApp, email

## Oeuvres

Les 16 images sont dans `CHAMASITE`, renommees de `oeuvre-01.jpeg` a `oeuvre-16.jpeg`.
Les titres artistiques, categories et descriptions sont dans `data.js`.

## Contacts a personnaliser

Dans `script.js`, remplacer :

- `whatsappNumber`
- `emailAddress`

Dans `contact.html`, remplacer aussi les liens WhatsApp et email visibles.
